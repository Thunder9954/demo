# **App Name**: CallEase

## Core Features:

- Call UI: Interactive interface for making and receiving calls.
- Call Recording: Record ongoing calls with user consent and proper notifications.
- Call History: Log and display past calls, including missed and received.
- Contacts Integration: Access and manage phone contacts.
- Number Blocking: Allow users to block unwanted numbers to filter incoming calls.
- Incoming Call Gesture: Use horizontal gesture for accepting or rejecting calls.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional, reliable feel.
- Background color: Very light blue (#E8EAF6).
- Accent color: Purple (#7E57C2) for highlighting interactive elements.
- Body and headline font: 'PT Sans', a sans-serif font with a modern yet warm feel suitable for both headings and body text.
- Use clear and intuitive icons for call actions and settings.
- Prioritize call controls with a clean, functional layout.
- Subtle animations to indicate call states and UI transitions.
